# 🩸 Sistema de Banco de Sangue

## 📌 Descrição do Projeto

O presente projeto tem como objetivo o desenvolvimento de um banco de dados destinado à gestão de um sistema de banco de sangue.

A aplicação busca organizar e automatizar os processos de coleta, armazenamento, processamento e distribuição de bolsas de sangue e seus componentes, associando cada bolsa a doadores e receptores com base em critérios como tipo sanguíneo, compatibilidade, validade e disponibilidade.

O sistema também contribui para a otimização dos estoques, segurança dos processos e rastreabilidade completa do ciclo da doação até a transfusão.

---

## 🎯 Objetivo

Desenvolver um sistema de banco de dados capaz de:

- Gerenciar doadores e receptores
- Controlar estoque de sangue por tipo sanguíneo
- Registrar doações e transfusões
- Atender requisições hospitalares
- Garantir rastreabilidade do sangue
- Otimizar a logística de distribuição

---

## 🧠 Funcionalidades do Sistema

- Cadastro de doadores
- Cadastro de receptores
- Registro de bolsas de sangue
- Controle de validade das bolsas
- Controle de estoque por tipo sanguíneo
- Registro de requisições hospitalares
- Histórico de doações e transfusões

---

## 🗄️ Tecnologias Utilizadas

- Modelagem de Dados (MER/DER)
